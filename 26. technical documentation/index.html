Introduction

HTML (HyperText Markup Language) is the standard language for creating web pages. It describes the structure of web pages using tags. Tags label pieces of content such as headings, paragraphs, links, and more.

Basic

An HTML document is made up of various parts:

  • DOCTYPE Declaration: Specifies the version of HTML used (e.g., <!DOCTYPE html>).
  • HTML Element: The root element that contains all other elements (<html>).
  • Head Section: Contains meta-information about the document (e.g., title, character set) and is defined by the <head> tag.
  • Body Section: Contains the content of the web page that is displayed to the user, enclosed in the <body> tag.

Below is an example of a simple HTML document:


        <!DOCTYPE html>
        <html>
          <head>
            <title>My First HTML Page</title>
          </head>
          <body>
            <h1>Hello World!</h1>
          </body>
        </html>
Elements

HTML elements are the building blocks of HTML pages. They consist of:

  • Opening Tag: Indicates the start of an element (e.g., <h1>).
  • Content: The text or other elements that are enclosed by the opening and closing tags.
  • Closing Tag: Indicates the end of an element (e.g., </h1>).

Common HTML elements:

  • Headings: <h1> to <h6> for different heading levels.
  • Paragraph: <p> for text paragraphs.
  • Links: <a> for hyperlinks.
  • Images: <img> for displaying images.
Attributes

Attributes provide additional information about HTML elements and are always specified in the opening tag. They come in name/value pairs, separated by an equals sign.

Example HTML attributes:

  • href: Specifies the URL of a link.
  • src: Specifies the source of an image.
  • alt: Provides alternative text for an image.

        <a href="https://www.w3schools.com/html/">
Forms

HTML forms are used to collect user input. They consist of various input elements, such as text fields, checkboxes, radio buttons, and buttons, all enclosed within a <form> tag.

Common form elements:

  • Text Input: <input type="text">
  • Password Input: <input type="password">
  • Submit Button: <input type="submit">
  • Select Dropdown: <select></select>

        <form action="/submit" method="post">
          <label for="name">Name:</label>
          <input type="text" id="name" name="name">
          <input type="submit" value="Submit">
        </form>
